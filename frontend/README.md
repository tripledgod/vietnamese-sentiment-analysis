# Frontend

Giao diện demo cho API phân tích cảm xúc.

## Chạy

Mở `index.html` trực tiếp trong trình duyệt, hoặc dùng static server:

```bash
# Python
python -m http.server 3000 --directory .

# Node (npx)
npx serve .
```

Mặc định kết nối tới `http://localhost:8001` (backend API). Có thể chỉnh API URL trong giao diện.

## Yêu cầu

- Backend API đang chạy tại port 8001
